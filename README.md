# Homebrew Tap for dlgochan

Personal Homebrew tap for custom formulas.

## Available Formulas

### claude-autoupdate

Automatic updates for claude-code Homebrew installations.

```bash
brew tap dlgochan/tap
brew install claude-autoupdate
brew claude-autoupdate install
```

See [claude-code-autoupdate](https://github.com/dlgochan/claude-code-autoupdate) for details.
